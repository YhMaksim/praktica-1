Нумерованный список

<ol>...</ol> - область нумерованного списка
<li>...</li> - тег элемента списка. Для каждого автоматически проставляется номер

Атрибуры:

type - вид счётчика
A - большие латинские буквы
a - маленькие латинские буквы
I - большие римские цифры
i - маленькие римские цифры
1 - обычные цифры
start - устанавливает число начала отсчёта
